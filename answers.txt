
Q1: Why might be consider using git?

Answer: Git enables collaboration and version controlled coding. It provides a backup solution and also allows the users to work in branches.

Q2: What field could data science be applied to? Give and Example and how it might help

Answer: Data science can be applied in healthcare for predictive analytics to improve patient care. By analyzing patient data, machine learning models can predict outcomes like disease development or hospital readmissions. This helps with early diagnosis, personalized treatments, and optimizing hospital operations for better efficiency.


Q3: What is the difference between merging and rebasing?

Answer: Merging combines the changes from one branch into another by creating a new merge commit. Rebasing moves the commits from one branch to another without creating a merge commit.
